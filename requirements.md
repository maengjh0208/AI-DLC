# 포모도로 타이머 개선 요구사항

## 1 단계: 타이머 설정 기능 수정
- 작업 시간을 5분~60분 사이에서 사용자가 조정 가능하게 (기본값 25분)
- 휴식 시간을 1분~30분 사이에서 사용자가 조정 가능하게 (기본값 5분)
- 설정 UI는 타이머 위에 배치
- 타이머 실행 중에는 설정 변경 불가능하게

---

## 2 단계: 작업 추적 기능 추가 
### 2.1 작업 정보 입력
- 타이머 시작 전 작업 이름 입력 필드
- 카테고리 선택 드롭다운 (개발, 디자인, 회의, 학습, 기타)
- 카테고리는 사용자가 추가/수정/삭제 가능하게

### 2.2 데이터 저장
- 각 세션마다 다음 정보 저장:
  * 작업 이름
  * 카테고리
  * 시작 시간
  * 종료 시간
  * 실제 작업 시간(분)
- localStorage 사용해서 브라우저에 저장

---

## 3 단계: 통계 기능 추가
### 3.1 통계 화면
- 타이머 화면과 별도 탭으로 분리
- 오늘/이번 주/이번 달/전체 기간 필터
- 카테고리별 총 작업 시간 (원형 차트)
- 개별 작업 목록 테이블 (정렬 가능)
  * 열: 날짜, 작업 이름, 카테고리, 작업 시간
- 총 작업 시간 요약 표시

---

## 4 단계: 캘린더 및 그래프 기능 추가
### 4.1 캘린더
- 월별 캘린더 뷰
- 각 날짜에 그날의 총 작업 시간 표시
- 날짜 클릭 시 해당 일의 상세 작업 내역 표시

### 4.2 그래프
- 기간 선택 (시작일~종료일)
- 카테고리별 누적 막대 그래프
- 일별/주별/월별 단위 전환 가능
- Chart.js 또는 간단한 CSS 막대 그래프 사용

### 4.3 필요한 라이브러리
- 그래프: Chart.js CDN 사용
- 깔끔한 디자인 유지

---

## 5. 추가 요청 기능
- 반응형 디자인으로 모바일에서도 잘 보이게
- 다크모드/라이트모드 토글 추가
- 작업 세션 삭제/수정 기능
- 데이터 내보내기 (CSV/JSON)
- 목표 시간 설정 및 달성률 표시
